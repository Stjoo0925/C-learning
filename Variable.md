# Csharp 변수의 대표적인 종류

- ```int``` : 정수형 변수타입 
	```csharp
	ex)
	int a = 10;
	```
- ```float``` : 실수형 변수타입
	```csharp
	float pi = 3.1415;
	```
- ```bool``` : 논리형 변수타입
	```csharp
	bool isGPSEnabled = true;
	```
- ```string``` : 문자열형 변수타입
	```csharp
	string myName = "Denis";
	```
- ```char``` : 문자형 변수타입
	```csharp
	char at = '@';
	```
